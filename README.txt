What dose this app do ?

This is a birthday reminder.
 If you want to get remind messages not to forget birthdays of people you love,
this app is the perfect for you.
when any birthday you registerd is within a week, it will send you a remind massage.




Before you use this app

1. need to have Gmail.

2. go to this link:
https://www.google.com/settings/security/lesssecureapps

And select: "Access for less secure apps"  otherwise this app 
doesn't work.

3. need to know that this app is a terminal app so, you need to "sign in" 
and "update" to get a remind massage.
